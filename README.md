COVID-19 Global Dashboard

Project Overview

The project is a data visualization dashboard implemented in Python using Pandas and Streamlit․
The tool analyzes global COVID-19 statistics and presents them with interactive charts and comparisons․

The dashboard also provides access to case trends‚ country comparisons‚ and the number of cases and deaths each day.

Features
* Country-wise COVID-19 analysis
* Total cases trend over time
* Graph of new daily cases
* Analyzing daily death trends
* Top 10 countries with highest cases
* Comparison of multiple countries

Technologies Used
* Python
* Pandas
* Streamlit
* Matplotlib (for visualization)
  
Dataset
The dataset used in this project contains daily global counts of COVID-19 cases:
* Date
* Country
* Total cases
* Daily new cases
* Active cases
* Total deaths
* Daily new deaths


How to Run the Project
Run the Streamlit app
streamlit run app.py OR pyhton -m streamlit run app.py

The dashboard will open in your browser.
